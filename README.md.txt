# Python Assignment - Part 1: Grade Tracker & String Utility

This repository contains the first part of the Python assignment, focusing on data parsing, basic control flow, and string manipulation.

## 📁 Project Structure
- `part1_grade_tracker.py`: The main Python script containing all 4 tasks.
- `README.md`: Project documentation (this file).

## 🚀 Tasks Implemented

### Task 1: Data Parsing & Profile Cleaning
- Cleaned raw student data (names, roll numbers, and marks).
- Validated names to ensure they contain only alphabetic characters.
- Converted data types (String to Integer/Float) for processing.

### Task 2: Marks Analysis
- Calculated the average marks for each student.
- Implemented conditional logic to assign "Pass" or "Fail" status based on a 50% threshold.

### Task 3: Class Performance Summary
- Generated a summary of total passed and failed students.
- Identified the class topper.
- Calculated the overall class average.
- Formatted output using f-strings.

### Task 4: String Manipulation Utility
- Processed a student essay using string methods like `.strip()`, `.title()`, and `.replace()`.
- Counted specific word occurrences and formatted the text into numbered sentences.
